YourSurprise

YourSurprise is een bedrijf wat gespecialiseerd is in het ontwikkelen van gepersonaliseerde cadeaus. Denk hierbij bijvoorbeeld aan een mok met een foto van je gezin of aan een glas waarin je naam staat gegrafeerd. Het bedrijf is opgericht in 2005 op een zolderkamer in Zierikzee met het ontwikkelen van deze cadeaus. Sindsdien zijn ze blijven groeien en exporteren ze hun cadeaus door heel Europa.

Werknemers van het bedrijf YourSurprise zeggen dat ze eigenlijk geen vaste taken hebben binnen hun specialistatie. Daarom is de sfeer in het bedrijf erg goed.

Beroepen binnen het bedrijf in de IT

  • PHP Developer
    • Een PHP developer is iemand die websites, applicaties en andere webtoepassingen ontwerpt, programmeert en implementeert in de programmeertaal PHP
  • JavaScript Developer
    • Een JavaScript developer is een webontwikkelaar die werkt met de scripttaal JavaScript

Technieken die gebruikt worden in het bedrijf binnen de IT

  • Artificial Intelligence
    • Word gebruikt voor het inschatten van de nog te verwachten bestellingen

Veracode

Veracode is een bedrijf wat gespecialiseerd is in het onwikkelen van beveiligingsapplicaties. Voorbeeld: je haalt je gemaakte code door een applicatie van Veracode en die stuurt een rapport terug naar jouw met alle beveiligings geralateerde fouten en met advies voor het oplossen hiervan. Het bedrijf is opgericht in 2006 in Burlington, Massachusetts (VS). Sinds de oprichting zijn de opgeschaald naar 4 kantoren in verschillende landen. Veracode biedt werknemers ook aan om vanuit huis te werken wanneer je in een geheel ander land woont.

Omdat er bij Veracode veel verschillende mensen uit veel verschillende landen werken is de cultuur heel verschillend. Een aantal keer per jaar worden werknemers uitgevlogen naar het hoofdkantoor om bij te praten.

Beroepen binnen het bedrijf in de IT (Veracode heeft er veel, hier een aantal)

  • Data Scientist
    • Een data scientist is iemand die grote hoeveelheden data verwerkt uit verschillende bronnen om deze om te zetten naar bruikbare informatie voor het managment van een bedrijf.
  • Software Engineer
    • Een software engineer is iemand die op basis van behoeften van de eindgebruiker software onwikkelt, schrijft, implementeert, test, verbetert en onderhoudt.
  • Front End Engineer
    • Een front end engineer is iemand die verantwoordelijk is voor de technische kant van het front-end (voorkant) van een applicatie, website of programma. Hij zorgt in feite voor de verbinding tussen het design en de programmatuur die nodig is om het te laten functioneren.
  • Backend Software Engineer
    • Een back end engineer is iemand die verantwoordelijk is voor de back-end (achterkant) van een website, applicatie of programma.
  • Software Architect
    • Een software architect is iemand die zich bezighoudt met het in kaart brengen, bewaken en optimaliseren van de structuur van software.

Technieken die gebruikt worden in het bedrijf binnen de IT

  • SaaS (Software as a Service)
    • Software die als online dienst wordt aangeboden. Dit zijn de meeste producten van Veracode.

Syntess Software

Syntess Software is een bedrijf wat zich specialiseerd in het maken en ontwikkelen van ERP-software (Enterprise Resource Planning). Dit is software wat gebruikt word binnen een organistatie ter ondersteuning van alle processen in het bedrijf. Een ERP-programma bestaat meestal uit kleine deelprgramma's die allemaal een specifieke taak ondersteunen. Syntess Software is opgericht in 1987 en zijn op dit moment gevestigd in Zierikzee. Hun ERP-software bieden ze aan als softwarepakketten.

De cultuur bij Syntess Software is werkt erg professioneel.

Beroepen binnen het bedrijf in de IT

  • Software Developer Team Web Apps
    • Een software developer is iemand die op basis van behoeften van de eindgebruiker software onwikkelt, schrijft, implementeert, test, verbetert en onderhoudt. Bij Syntess is de software developer gericht op het maken van Team Web Apps.
  • Front-End Developer
    • Een front end developer is iemand die verantwoordelijk is voor de technische kant van het front-end (voorkant) van een applicatie, website of programma. Hij zorgt in feite voor de verbinding tussen het design en de programmatuur die nodig is om het te laten functioneren.
  • Software Ontwikkelaar
    • Een software ontwikkelaar is iemand die op basis van behoeften van de eindgebruiker software onwikkelt, schrijft, implementeert, test, verbetert en onderhoudt.

Technieken die gebruikt worden in het bedrijf binnen de IT

  • Protocol UPI (Uniform Process Integration)
    • Syntess Software gebruikt dit om alle werkprocessen in één softwarepakket te stoppen.

Macaw

Macaw is een bedrijf dat verschillende digitale oplossingen levert voor andere bedrijven. Een bedrijf legt een probleem op tafel bij Macaw en zij zorgen dat ze het oplossen met digitale software. Naast het implementeren en onderhouden van technologie, helpt Macaw ook bij het strategisch plannen, designen, de adoptie, doorontwikkeling en de optimalisatie.

Werknemers van Macaw geven aan dat hun bedrijf erg casual/professioneel is.

ASML

ASML is een Nederlands bedrijf wat gespecialiseerd is in het maken van chips. Het bedrijf bestaat al sinds 1984 en zijn op dit moment gevestigd in Veldhoven.

Beroepen binnen het bedrijf in de IT

  • Alogrithm Engineer
    • Een algoritme-ontwikkelaar is een technische professional die in het algemeen verantwoordelijk is voor het ontwikkelen van algoritmen, die technische stukjes computercode zijn die specifieke resultaten op veel verschillende gebieden produceren.
  • Data Scientist
    • Een data scientist is iemand die grote hoeveelheden data verwerkt uit verschillende bronnen om deze om te zetten naar bruikbare informatie voor het managment van een bedrijf.
  • Software Engineer
    • Een software engineer is iemand die op basis van behoeften van de eindgebruiker software onwikkelt, schrijft, implementeert, test, verbetert en onderhoudt.
  • Software Design Engineer
    • De taak van de software designer engineer is het opzetten van een bepaald plan voor een bepaalde behoefte. Wanneer een bedrijf een programma nodig heeft dat niet op de markt verkrijgbaar is en daarvoor een ICT-bedrijf inhuurt, moet de software designer engineer tot in de puntjes begrijpen wat de wensen van de klant zijn, en moet hij hier vervolgens een plan en methode voor ontwikkelen.
  • Data Leakage Prevention Specialist
    • Je bent verantwoordelijk voor het monitoren en analyseren van incidenten op het gebied van Data Leakage Prevention (DLP).
